MERN Food Delivery Web Application

A full-stack food delivery web application built using the MERN stack with authentication, role-based access, cart system, and order management.

---

# Features

### User
- User registration & login (JWT authentication)
- Role-based access (User / Admin)
- Browse food items
- Add to cart & checkout
- Order history
- Order status tracking
- Dark mode support

### Admin
- Admin dashboard
- Add / edit / delete food items
- View all user orders
- Update order status
